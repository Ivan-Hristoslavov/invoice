# Fix for the Stripe API key issue

The error "Neither apiKey nor config.authenticator provided" is occurring because your 
STRIPE_SECRET_KEY in the .env file is malformed. There's a comment directly attached to the 
key value which is causing Stripe to fail.

```

STRIPE_SECRET_KEY_FIXED=sk_test_51QdfKuRubUJH2vI3ZUu3fjDqvxiL8fQqLQrqGApw8D4Q1pVxgHLK8aAxwdOr2bPmVZWc8ORkdFILX9NdXOfPk6z5006S9FLylG
```
