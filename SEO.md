## Следващи стъпки за по-добро SEO

### 1. Google Search Console
- Регистрирайте сайта в Google Search Console
- Добавете verification code в `app/layout.tsx`
- Изпратете sitemap.xml

### 2. Google Analytics
- Добавете Google Analytics 4
- Настройте conversion tracking
- Следете user behavior

### 3. Съдържание
- Добавете блог секция с полезни статии
- Създайте FAQ страница
- Добавете case studies и testimonials

### 4. Backlinks
- Регистрирайте в български бизнес директории
- Създайте профили в социални мрежи
- Публикувайте в релевантни форуми

### 5. Local SEO
- Google Business Profile (ако имате офис)
- Регистрация в български бизнес портали
- Локални ключови думи

### 6. Performance Monitoring
- Използвайте PageSpeed Insights
- Оптимизирайте Core Web Vitals
- Мониторирайте loading times

### 7. Social Media
- Създайте Facebook страница
- LinkedIn профил за компанията
- Twitter/X акаунт

## Environment Variables

Добавете в `.env.local`:
```
NEXT_PUBLIC_APP_URL=https://facturapro.bg
```

Използвайте следните инструменти:

- Google PageSpeed Insights
